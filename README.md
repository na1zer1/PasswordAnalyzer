# PasswordAnalyzer

## Introduction

PasswordAnalyzer is a CLI application for analyzing and evaluating password strength, written in Python and Rust.

This is my first independent programming project :)

## 🤷‍♂️Installation

Before using the application, make sure you have Rust installed.
Download Rust from the official website:
https://rust-lang.org/ru/tools/install/

Run the installer and select: `Standard Installation`
Wait until the installation is complete.

You also need Python installed on your system:
https://www.python.org/downloads/

After installing Python, open Command Prompt and install the required dependencies:
```cmd
pip install rich
```

## 🚀 Usage

After installing all dependencies, run the application using your preferred IDE or from the terminal.

## 🛠Features

* Password analysis with a score from 0 to 100
* Password history viewing
* Enable or disable password history
* Clear password history
* Password improvement recommendations
* Password generation

## 💻Technologies

* Python
* Rust
